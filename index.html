Science Made Easy Videos

The following content is part of the "Science Made Easy" DVD developed by Prof. Henry Brouwer, emeritus professor at Redeemer University. The orginal DVDs incuded additional materials and documents which included a table of contents, a summary of each demonstration, an introduction, a forward, a table listing some common chemicals, and a set of safety guidelines.

Before proceeding with any of these videos, please refer to the Safety Guidelines. It is the user's responsibility to follow proper safety procedures when doing any science activity.
